📍 地政学の5つの核心原則
1️⃣ 決定論的地理観
「地理は『選択肢の幅』を制限する」
  • 内陸国（チェコ、オーストリア） → 必然的に陸上大国と付き合わざるを得ない
  • 島国（英国、日本、台湾） → 海洋国家として発展するか、海洋国に支配されるかの2択
  • 地中海国（イタリア、ギリシャ） → ナイル川域国と競争
2️⃣ パワー分布の『不変性』
「大国の相対的位置は『世紀単位』で変わらない」
  • 米国：地理的孤立 → 200年間、北米での相対的支配継続
  • ロシア：広大な平原 → 常に周辺国への脅威（ナポレオン、WW2、ウクライナ侵攻）
  • 中国：人口13億 + 大陸性 → 東アジアの覇権国家は『避けられない』
3️⃣ 隣国との『自動競争』
「理由がなくても、地理的に隣接する国は競争する」
  • 国境確保 → 影響力拡大 → 隣国の脅威 → 軍備増強 → 対立悪化
  • この悪循環は『理性的判断』では止められない（地理による『構造的必然』）
4️⃣ リソース・チョークポイントの戦略的価値
「海峡・鉱山・水源が国家戦略を左右する」
  • マラッカ海峡（シンガポール）：世界の石油の 30% → シンガポールの影響力は絶大
  • ホルムズ海峡（イラン・オマーン）：世界の石油の 20% → イランの『核カード』の源
  • 南シナ海：石油 + レアアース + 世界貿易の 30%
5️⃣ 『バランス・オブ・パワー』の永遠の追求
「国際政治は『力の均衡』が安定を生む」
  • 一つの国が『支配的』になると、他国が結合して抵抗する
  • 例：ナポレオン → ウィーン会議でバランス回復
  • 例：ヒトラー → WW2 でアメリカ・ロシア・イギリス連合でバランス回復
🎓 歴史的な地政学の3大学派
マッキンダー『ハートランド論』（1904年）
「ユーラシア内陸部の支配 = 世界支配」
東ヨーロッパ の支配 → ハートランド（ロシア）の支配 → 世界覇権
根拠：
  • ユーラシア大陸は世界最大の陸上資源を保有
  • ロシアが統一されると、西ヨーロッパは対抗できない
  • だから西欧（イギリス）は東ヨーロッパを『バッファーゾーン』として保つべき
現代的意義： プーチンのウクライナ侵攻は『まさにこの論文を実行している』
スピークマン『リムランド論』（1944年）
「マッキンダー反論。大陸『周辺部』（海洋国）が世界を支配する」
島国（米国）が大陸周辺部を支配 → ハートランド孤立化 → 米国覇権
戦略：
  • 第1島鎖：日本 → 台湾 → フィリピン → インドネシア
  • 第2島鎖：グアム → パラオ → オーストラリア
  • これらを『米国の勢力圏』として保つ → 中国とロシアを『陸上に閉じ込める』
現代的意義： 米国の『インド太平洋戦略』はまさにこれ
ブレジンスキー『グランド・チェスボード』（1997年）
「21世紀の米国戦略はユーラシアの『複数の地域大国』を支配することだ」
NATO拡大 → EU統合 → 日本・韓国強化 → インド台頭 → 中国・ロシア包囲網
具体的タクティクス：
  1. NATO を東ヨーロッパに拡大（ロシア西側を米国側へ）
  2. 日本・台湾を強化（東アジアを米国側へ）
  3. インドの台頭を支援（南アジアを米国側へ）
  4. シーレーン（海上交易路）を米国海軍が支配
現代的意義： 現在の『Quad（クアッド）』が戦略の実装
